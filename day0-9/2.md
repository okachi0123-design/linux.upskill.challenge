## What I Learned
- `pwd`
- `cd`
- `ls`
- `ls -ltr`で古い順に並べて、最近変更したファイルを一番下に表示できる
- `man`
- `man -k `キーワード検索
- `apropos `キーワードからコマンドを検索　networkでネットワーク関連のコマンドリストと使い方の表示など
- `man`が効かないコマンドは**builtin（シェル組み込み）コマンド**で、その場合は`help`を使う
- `pushd`/`popd`でディレクトリ履歴をスタック形式で管理できる
- 環境変数`PS1`でシェルプロンプトの表示をカスタマイズできる

## Commands Used
```bash
# ナビゲーション
pwd                  
cd                     
pushd /var/log         # ディレクトリ移動（履歴に積む）
popd                   # 履歴から戻る

# ファイル操作
ls                     
mkdir 

# マニュアル
man
apropos # キーワードからコマンド検索
man -k # aproposと同じ

```
