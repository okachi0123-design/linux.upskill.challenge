## What I Learned
- Linuxのユーザーは3種類:`root`(全権限)、`sudoers`(sudoで昇格可能な一般ユーザー)、`regular users`(自分の領域のみ)
- **rootで直接ログインしない**
- **Global変更**(全ユーザーに影響)と**Local変更**(特定ユーザーのみ)の区別が重要
- `/etc/shadow`はパスワードハッシュを保存するファイル。
- `hostnamectl`ホスト名変更
- `timedatectl`でタイムゾーン変更
- `sudo -i` `sudo -s` 
  - `-i`:rootとしてログインし直す感覚。環境変数もホームディレクトリもrootのものに切り替わる
  - `-s`:シェルだけrootに昇格。環境は現在のユーザーのまま
## Commands Used
```bash
# sudo基本
sudo cat /etc/shadow       # 通常は読めないファイルもsudoで読める
sudo -i                    # rootとしてログインシェルを起動
sudo -s                    # 現在の環境のままrootシェルに昇格
exit                       # sudo -i / sudo -s から抜ける

# ログイン履歴
last                       # 全ユーザーのログイン履歴
last <username>            # 特定ユーザーのみ
sudo lastb                 # 失敗したログイン試行

# sudo使用履歴
sudo journalctl -e /usr/bin/sudo

# 再起動・稼働時間
sudo reboot
uptime

# ホスト名変更
sudo hostnamectl set-hostname mylittlecloudbox

# タイムゾーン確認・変更
timedatectl
timedatectl list-timezones
sudo timedatectl set-timezone Asia/Tokyo
```
