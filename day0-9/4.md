## What I Learned
- `apt search`パッケージ検索
- `apt install`インストール
- 主要な設定ファイルは`/etc`配下
- ログは`/var/log`配下
- `mc`(Midnight Commander)はテキストベースの2画面ファイルマネージャ
- 重要な設定ファイル
  - `/etc/passwd`:ユーザー情報(パスワードハッシュは含まない)
  - `/etc/ssh/sshd_config`:SSHサーバーの設定
  - `/var/log/auth.log`:認証関連のログ
  - `q`keyで設定ファイルから出れる
- `/etc/passwd`誰でも読めるユーザー情報
- `/etc/shadow`rootのみ読めるパスワードハッシュ
## Commands Used
```bash
# パッケージ管理
sudo apt update                       # パッケージリストを更新
sudo apt search <keyword>             # パッケージを検索
sudo apt install <package>            # インストール
sudo apt remove <package>             # アンインストール
apt show <package>                    # パッケージの詳細情報
# mc(Midnight Commander)
sudo apt install mc                   # mcをインストール
mc                                    # 起動
# mc内でF3=ファイル閲覧、F10=終了
# 設定ファイル・ログを覗く
cat /etc/passwd                       # ユーザー一覧
sudo cat /etc/ssh/sshd_config         # SSH設定
sudo tail /var/log/auth.log           # 認証ログ
```
